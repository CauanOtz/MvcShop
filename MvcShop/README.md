# MvcShop (Exemplo MVC)

Projeto exemplo .NET 6 MVC com:

- Home com carrossel
- Links para Contatos e Produtos
- Vitrine de produtos separados por categoria
- Página de detalhes do produto
- Área Admin com CRUD básico de Produtos e Categorias

Como rodar:

```powershell
cd "c:\Users\cicer\OneDrive\Documentos\FATEC\MVC_DESIGNPATTERN\MvcShop"
dotnet restore
dotnet run
```

O projeto usa banco em memória para demonstração (EF Core InMemory).
